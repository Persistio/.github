# Persistio

**Smarter agent conversations through enhanced memory extraction and recall.**

Persistio is a self-hosted, API-first memory service for AI agents. It automatically extracts meaningful context from conversations and surfaces the right memories at the right time — so your agents remember what matters.

---

## What it does

- **Automatic extraction** — ingests conversation history and pulls out durable facts, preferences, and decisions
- **Semantic recall** — retrieves relevant memories at query time using vector search
- **API-first** — clean REST API, easy to integrate with any agent framework
- **Self-hosted** — run it on your own infrastructure, keep your data yours
